Applying Attribute Grammars to teach Linguistic Rules

Using the formalisms of attribute grammars, the main goal of this study is to create a tool to help linguistics students learn and study the different rules of natural language.

  • Build a DSL with a simple notation such that any person that does not have experience with common programming language elements could quickly use and grasp.
  • Connect this notation with a visual component to allow the visualization of different natural language syntax trees.
